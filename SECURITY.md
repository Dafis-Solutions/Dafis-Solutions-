# سياسة الأمان (Security Policy)

## الإصدارات المدعومة
نحن ندعم التحديثات الأمنية للإصدارات التالية:

| الإصدار | مدعوم بالتحديثات الأمنية |
| ------- | ------------------------ |
| v1.0.x  | نعم                   |
| < v1.0  | لا                    |

## الإبلاغ عن ثغرة أمنية
إذا اكتشفت ثغرة أمنية في هذا الموقع، يرجى عدم فتح مشكلة (Issue) عامة. بدلاً من ذلك، يرجى التواصل معنا مباشرة عبر البريد الإلكتروني الخاص بإدارة المشروع ليتم التعامل معها بشكل آمن وسري.
